# NVIDIA USD Composer 快速入門指南

本文件旨在協助開發者快速掌握 **NVIDIA USD Composer (原 Omniverse Create)** 的基礎操作，包含環境配置、物件建立與場景導航邏輯。

---

## 🛠 一、 工作空間優化 (Setting Up)

在進行 USD 創作前，優化 UI 佈局能顯著提升工作效率。

1. **新建場景**：啟動應用程式後，選擇 `File` > `New`。
2. **面板佈置**：
   * 建議將 **Stage (場景)** 面板拖曳至 **Layer (圖層)** 面板旁並排顯示。
   * **工程師觀點**：這能讓你同時監控「場景階層 (Structure)」與「圖層覆蓋 (Deltas)」，這是 USD 工作流的核心。

---

## 📦 二、 物件建立流程 (Object Creation)

練習建立基礎幾何物件（Mesh）以熟悉介面操作：

* **操作路徑**：`Create` > `Mesh` > `Sphere` (球體)。
* **結果**：預設球體將會生成於世界座標原點 $(0, 0, 0)$。

---

## 🚀 三、 場景導航與視角控制 (Navigation)

掌握「飛行模式」是 3D 場景編輯的基本功。

### 1. 基礎滑鼠操作
| 操作 | 動作 |
| :--- | :--- |
| **旋轉 (Orbit)** | 長按 **滑鼠右鍵** + 移動滑鼠 |
| **平移 (Pan)** | 長按 **滑鼠中鍵** + 移動滑鼠 |

### 2. 飛行模式 (WASD 控制)
在 **長按滑鼠右鍵** 的狀態下，配合鍵盤按鍵：

| 按鍵 | 功能 |
| :--- | :--- |
| **W / S** | 前進 / 後退 |
| **A / D** | 左平移 / 右平移 |
| **Q / E** | 下降 / 上升 |
| **CTRL + 按鍵** | **微速飛行** (用於精確對齊視角) |

### 3. 快速聚焦 (Focus)
若在廣大場景中遺失目標，請使用：
* **快捷鍵 `F`**：在 Stage 面板選中物件後按下，視角將立即對準該物件。

---

## 🌲 四、 場景階層管理 (Stage & Prims)

在 USD Composer 中，所有元素皆被視為 **Prim (Primitive)**。

* **World (defaultPrim)**：場景的根節點。
* **Environment**：包含預設的環境光（Sky）、遠程光（DistantLight）及地面（ground）。
* **Mesh / Looks**：存放模型幾何資料與材質定義。

> [!TIP]
> **開發小筆記**：
> 如果你發現畫面一片漆黑或物件不見了，先檢查 **Stage** 面板中物件的「眼睛」圖示是否開啟，並善用 **`F`** 鍵重置視角。
